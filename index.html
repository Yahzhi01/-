中级汉语写作提升平台

HSK4–5 写作教学设计展示（IPO + ZPD）

🎯 Can-Do学习目标

学习者能够在提供写作结构模板和功能表达支架的条件下， 规划并构建一篇约120字的汉语短文， 并在系统反馈的引导下对语法错误、语序问题和词汇使用进行 诊断与修正， 进一步对文本表达效果进行评估与优化， 完成更为清晰、连贯的二次改写。

🔁 IPO任务链

输入 Input AI范文 + 写作模板 + 表达支架 → 建立结构与表达认知

处理 Process 结构识别 → 提纲构建 → 写作生成 → 修改加工

输出 Output 完成短文 → 根据反馈优化 → 二次改写

🧩 分层支架设计（ZPD）

A类（初级）
高支架：词汇表 + 句型模板 + 填空写作 + 拼接句子
👉 目标：建立基本句法结构与三段式意识
B类（中级）
中支架：关键词提示 + 自主提纲 + 系统反馈修改
👉 目标：提升连贯性与表达准确性
C类（进阶）
低支架：仅提供任务情境 + 自主写作 + 自我评估
👉 目标：发展表达多样性与语言灵活性

✍ 写作训练（教学闭环）

结构提示：背景 → 原因 → 总结
